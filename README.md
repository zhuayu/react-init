# react init

依赖包

- [reset-css](https://meyerweb.com/eric/tools/css/reset/) 重置样式
- [styled-components](https://www.styled-components.com/docs) 组件样式
- redux react-redux 数据管理
- [immutable.js](https://github.com/facebook/immutable-js)  管理 reducer 中 state 数据
- redux-thunk 让我们可以在reducer 的 action 中写函数，处理异步的请求逻辑
- react-router-dom 前端路由
- [react-loadable](https://github.com/jamiebuilds/react-loadable) 组件异步加载
- axios 请求

插件

- [redux-devtools-extension](http://extension.remotedev.io/) redux 数据调试
